DRR-AI Background

Restricted-Use Data

Richer Data, Deeper Insights

  • Restricted-use datasets from IES and ICPSR contain richer information than public-use data — including detailed individual characteristics, transcripts, and geographic identifiers.
  • This richness enables education researchers to investigate more nuanced and substantive questions than public-use files allow.
  • Because of this sensitivity, all products using restricted-use data must pass a Disclosure Risk Review (DRR) before publication or dissemination.

DRR-AI Our Solution

Introducing DRR-AI

Built for the Education Community

  • DRR-AI was created to support researchers and reviewers in the disclosure risk review process — reducing delays, errors, and re-submissions.
  • In compliance with restricted-use data license policies, DRR-AI uses a zero-retention architecture — all reviews are processed entirely in memory and no user data is ever retained.
